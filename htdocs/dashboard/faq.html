Windows Frequently Asked Questions

XAMPP for Windows exists in three different flavors: Installer, ZIP, and 7zip. The installer is the easiest way to install XAMPP. To use ZIP or 7zip, simply extract the files and run the setup_xampp.bat file.
Since XAMPP 5.5.30 and 5.6.14, XAMPP ships with MariaDB instead of MySQL. The commands and tools are the same for both.
If you extract XAMPP in a top-level folder like "C:\" or "D:\", you can start most servers like Apache or MySQL directly without running setup_xampp.bat. Using relative paths in the setup script is preferred if installing XAMPP on a USB drive.
The XAMPP Control Panel is the universal control center for starting/stopping services such as Apache, MySQL, FileZilla, and Mercury. You can also use batch files like xampp_start.exe and xampp_stop.exe.
Open your browser and type http://localhost/ or http://127.0.0.1/. If everything is set up correctly, you should see the XAMPP start page.
Each XAMPP server (Apache, MySQL, FileZilla) can be installed as a Windows service from the XAMPP Control Panel. Make sure to run the Control Panel as an Administrator when installing services.
XAMPP is not intended for production use but for development environments. It is open and lacks security measures necessary for a live environment. Ensure you secure the installation before putting it online.
If you installed XAMPP via the installer, use the Uninstaller program to remove it. For ZIP and 7zip installations, stop all services and delete the entire XAMPP directory.
To configure XAMPP for email, edit the php.ini and sendmail.ini files to configure the SMTP server. Restart Apache after making changes.
Open the XAMPP Shell and run the following command: mysqladmin.exe -u root password [yourpassword]. This sets the root password.
