Backup and Restore MySQL/MariaDB Databases

Using phpMyAdmin

XAMPP includes phpMyAdmin, a browser-based tool for managing MySQL/MariaDB databases. Follow these steps to backup a database:

  1. Browse to http://localhost/phpmyadmin and log in (default is root with no password).
  2. Select your database (e.g., bitnami_wordpress).
  3. Click "Export" in the top navigation.
  4. Select "Quick" export and "SQL" as the format, then click "Go" to download the backup file.

This will download a text file with SQL statements for your backup.

Restore with phpMyAdmin

  1. Browse to http://localhost/phpmyadmin and log in.
  2. Create a new database, e.g., myblog.
  3. Select the "Import" option and upload your backup file.
  4. Click "Go" to restore the database.

Using Command-Line Tools

You can also backup and restore using command-line tools:

Backup with mysqldump

mysqldump --user=root --password="" bitnami_wordpress > bitnami_wordpress.sql

This exports the database to bitnami_wordpress.sql.

Restore from Command Line

mysql --user=root --password="" --database=myblog < bitnami_wordpress.sql

This command restores your backup file to the myblog database.

Note: Adjust the file size limits for uploads in php.ini if your database is large.
